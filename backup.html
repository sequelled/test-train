Database Backup Solutions

Nothing is worse than losing your client's data, so having a well planned backup procedure is of utmost importance.


We can setup one or more methods to automatically backup your database clusters that meets your budget and requirement. Some of the options are:

  • Logical backup (Reliability - high, Restore time - high, Setup cost - low, PITR - no)
  • File system level backup (Reliability - high, Restore time - low, Setup cost - medium, PITR - no)
  • Continuous archiving and point in time recovery (Reliability - high, Restore time - medium, Setup cost - high, PITR - yes)
  • Backup monitoring
